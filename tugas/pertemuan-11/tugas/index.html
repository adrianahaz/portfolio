Apa itu Embedded CSS?

**Embedded CSS** (atau Internal CSS) adalah metode penulisan kode CSS yang diletakkan di dalam tag <style>. Tag ini biasanya ditempatkan di dalam bagian <head> pada dokumen HTML.

Contoh Penulisan:

<head>
  <style>
    body { background-color: linen; }
    h1 { color: maroon; }
  </style>
</head>

Kapan Menggunakannya?

  • Ketika Anda hanya ingin memberikan gaya pada **satu halaman spesifik**.
  • Jika Anda tidak ingin membuat file .css eksternal yang terpisah.
  • Untuk demo cepat atau pengujian kode (seperti halaman ini!).
