Apa itu External CSS?

**External CSS** adalah metode di mana kode CSS ditulis di file terpisah dengan ekstensi .css, lalu dihubungkan menggunakan tag <link> di dalam <head>.

Cara Menghubungkan:

<head>
  <link rel="stylesheet" href="style.css">
</head>

Keuntungan External CSS:

  • **Reusability**: Satu file CSS bisa digunakan untuk banyak halaman HTML.
  • **Organisasi**: Kode HTML jadi lebih bersih dan fokus pada struktur konten.
  • **Caching**: Browser menyimpan file CSS sehingga loading website lebih cepat saat pindah halaman.
